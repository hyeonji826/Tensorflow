## 프로젝트 소개
TensorFlow는 구글에서 개발한 오픈소스 머신러닝 프레임워크로, 다양한 딥러닝 모델을 구현하고 학습시키는 데 사용됩니다. 이 프로젝트는 TensorFlow의 기초부터 고급 개념까지 학습하고, 관련 자격증을 준비하는 과정을 기록한 것입니다.

이 리포지토리에서는 TensorFlow를 이용한 모델 구축, 실습 프로젝트, 주요 개념 등을 다루고 있습니다.

## 학습 과정
이 프로젝트는 TensorFlow의 기본적인 개념부터 심화 내용까지 포함한 학습 과정을 기록합니다. 각 주제는 이론 설명과 함께 실습 코드가 제공되어, 학습한 내용을 바로 실습할 수 있도록 돕습니다.

### 주요 학습 내용:
- **TensorFlow 설치 및 설정**: TensorFlow 환경 설정, 가상 환경 만들기
- **기본 개념**: 텐서, 변수, 세션
- **딥러닝 모델**: 신경망, CNN(Convolutional Neural Networks), RNN(Recurrent Neural Networks)
- **모델 학습**: 손실 함수, 옵티마이저, 에포크
- **모델 평가 및 튜닝**: 성능 평가 지표, 하이퍼파라미터 튜닝
- **실습 프로젝트**: MNIST 데이터셋을 사용한 분류 모델, 이미지 분류, 자연어 처리(NLP) 모델

## 실습 예제
- **MNIST 손글씨 분류**: 딥러닝을 사용하여 MNIST 데이터셋을 분류하는 모델을 구현합니다.
- **CIFAR-10 이미지 분류**: CIFAR-10 데이터셋을 사용하여 10가지 클래스의 이미지를 분류하는 모델을 학습시킵니다.
- **자연어 처리(NLP)**: 텍스트 데이터를 기반으로 하는 분류 모델을 개발하여, 텍스트 데이터를 분석하는 방법을 배웁니다.
- **모델 최적화**: 모델 성능을 개선하기 위한 다양한 기법을 실습합니다.
